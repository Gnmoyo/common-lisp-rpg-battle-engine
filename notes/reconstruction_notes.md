# Reconstruction Notes

This repository contains a portfolio version of a Common Lisp combat engine originally developed during internship work.

## What this repository shows

The purpose of this repository is to present:
- Common Lisp programming
- CLOS-based object-oriented design
- mixins and multiple inheritance
- generic function based combat logic
- test-oriented development practices

## Changes made for portfolio presentation

The original code was reorganized for readability and portfolio presentation by:
- placing the main implementation in a dedicated source file
- removing editor-specific comments
- adding project documentation
- adding examples and design notes

The goal is to preserve the original design ideas while presenting the project clearly and professionally.
